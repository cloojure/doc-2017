row-major-seq-no0d
(row-major-seq-no0d m)
Like row-major-seq but drops NDArray's wrapping on 0d-slices, so
`(row-major-seq-no0d (ndarray [1 2 3]))` will return sequence
`1 2 3` instead of a sequence of 0d NDArrays
